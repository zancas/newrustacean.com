pub fn demonstrate_cow(_point: &B_JustClonePoint)

The Cow type can wrap around other types and make them "reusable".

Note that the body of this function is identical with that of the body of the example below.

Examples

We'll reuse the B_JustClonePoint since Cow::Owned requires that the underlying type implement Clone.

let a_point = Cow::Owned(B_JustClonePoint::origin());
demonstrate_cow(&a_point);

Note that even though demonstrate_cow takes a reference to B_JustClonePoint, we can pass it the Cow instance; this is where the Deref implementation on Cow comes in handy.
