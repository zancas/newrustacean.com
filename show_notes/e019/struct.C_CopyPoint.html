A struct with identical behavior to A_NoCopyOrClonePoint, except with Copy.

Note that we have Clone as well as Copy here---we have to, since Clone is a supertrait for Copy.

Examples

Note that this is just like the non-compiling example in A_NoCopyOrClonePoint, but because C_CopyPoint implements Copy, the line which previously caused a compile error now works without any issue.

let a_point = C_CopyPoint::origin();
let copied_point = a_point;
println!("{:?}", a_point);  // <- not a problem
println!("{:?}", copied_point)
